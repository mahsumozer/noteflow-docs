Noteflow MCP#

Noteflow MCP gives AI tools like Claude Desktop, Cursor, and VS Code direct access to your Noteflow account. You can interact with your notes, projects, and tasks in natural language.

“Summarize the notes I created last month with the ‘meeting’ tag.” “Create a new project called Q3 Planning and add three tasks.” “Find my voice recordings about API design.”

Installation#

npm install -g noteflow-mcp

Or run it directly without installing (npx always downloads the latest version):

npx noteflow-mcp --help

Getting Started in 3 Steps#

1 — Get a Token#

Open the Noteflow app → Settings → Developer → press the Copy API Token button.

The token is a private key unique to your account. Do not share it with anyone.

2 — Add to Claude Desktop#

Open ~/Library/Application Support/Claude/claude_desktop_config.json (on Windows: %APPDATA%\Claude\claude_desktop_config.json) and add the following:

{
  "mcpServers": {
    "noteflow": {
      "command": "npx",
      "args": ["-y", "noteflow-mcp"],
      "env": {
        "NOTEFLOW_TOKEN": "buraya_tokenini_yapıştır"
      }
    }
  }
}

Save the file and restart Claude Desktop.

3 — Try It#

Ask Claude:

“Show my last 5 notes in Noteflow”

If everything is working, Claude will list your notes.

Cursor#

Add to your .cursor/mcp.json file:

{
  "mcpServers": {
    "noteflow": {
      "command": "npx",
      "args": ["-y", "noteflow-mcp"],
      "env": {
        "NOTEFLOW_TOKEN": "buraya_tokenini_yapıştır"
      }
    }
  }
}

VS Code (GitHub Copilot)#

Add to your .vscode/mcp.json file:

{
  "servers": {
    "noteflow": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "noteflow-mcp"],
      "env": {
        "NOTEFLOW_TOKEN": "buraya_tokenini_yapıştır"
      }
    }
  }
}

Which features are enabled?#

By default, only note reading (notes:read) is enabled. To enable more features, add NOTEFLOW_FEATURES:

"env": {
  "NOTEFLOW_TOKEN": "...",
  "NOTEFLOW_FEATURES": "notes:read,notes:write,projects:read,tasks:read,voice:read"
}

Full list of features → Feature Flags

Troubleshooting#

I’m getting a token error Sign out of the Noteflow app and sign back in, then copy a new token.

Claude doesn’t see the tools Save the config file, then fully close and reopen Claude Desktop.

I only want to see certain tools You can select the tools you want with NOTEFLOW_FEATURES → Feature Flags
